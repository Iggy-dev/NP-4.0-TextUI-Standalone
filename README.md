# NP-4.0-TextUI-Standalone

# Example Exports

<h3>Show Text UI</h3>

```lua
---usage: message/type
exports['TextUI']:Show('E', 'Example Text')
exports['TextUI']:Show('G', 'Example Text')
exports['TextUI']:Show('H', 'Example Text')
```

<h3>Hide Text UI</h3>

```lua
exports['TextUI']:Hide()
```
